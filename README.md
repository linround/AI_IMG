# AI_IMG
一款基于AI成图的图片处理工具。通过调节图片参数，即可生成相应的图片效果。

## 搭建自己的 AI 图床

- https://huggingface.co/BotsOne/utilitypole

- https://huggingface.co/ByteDance/SDXL-Lightning

## commit
- `feat`：新功能（feature）
- `fix`：修补bug
- `docs`：文档（documentation）
- `style`： 格式（不影响代码运行的变动）
- `refactor`：重构（即不是新增功能，也不是修改bug的代码变动）
- `test`：增加测试
- `chore`：构建过程或辅助工具的变动
